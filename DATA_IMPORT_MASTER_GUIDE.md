# 📚 吉林大学课程表应用 - 数据导入总指南

## 🎯 概述

本应用提供**两种**课表数据导入方式，您可以根据实际情况选择：

---

## 方式1️⃣：应用内WebView导入（自动）

**适用场景**：课表页面HTML结构标准

### 操作步骤
```
应用 → 设置 → 重新获取课表
     → 登录 i.jlu.edu.cn
     → 进入"我的课表"
     → 点击 📥 提取
```

### 特点
- ✅ 全自动，无需电脑
- ✅ 一键完成
- ❌ 依赖HTML结构
- ❌ 可能因页面更新失效

### 调试工具
- 🐞 **调试按钮**：查看页面元素统计
- 📊 **错误提示**：显示找到的课表元素数量

### 详细文档
- 📖 [HOW_TO_IMPORT_DATA.md](HOW_TO_IMPORT_DATA.md) - 完整指南
- 📖 [QUICK_IMPORT_GUIDE.md](QUICK_IMPORT_GUIDE.md) - 快速参考

---

## 方式2️⃣：JSON数据导入（推荐）⭐

**适用场景**：WebView无法提取、需要编辑数据、跨设备导入

### 操作步骤

#### 第一步：电脑提取（5分钟）
```
1. 电脑浏览器打开 i.jlu.edu.cn
2. 登录并进入课表页面
3. 按 F12 打开控制台
4. 粘贴提取脚本并运行
5. JSON自动复制到剪贴板
```

#### 第二步：传输到手机（2分钟）
```
微信/邮件/云盘 → 发送JSON到手机
```

#### 第三步：应用导入（1分钟）
```
应用 → 设置 → 导入课表JSON
     → 粘贴数据
     → 点击导入
```

### 特点
- ✅ 可靠稳定，不受HTML结构影响
- ✅ 可以手动编辑数据
- ✅ 支持数据备份和分享
- ✅ 适用于任何课表系统
- ❌ 需要电脑操作
- ❌ 步骤相对较多

### 详细文档
- 📖 [JSON_IMPORT_GUIDE.md](JSON_IMPORT_GUIDE.md) - 快速三步法 ⭐⭐⭐⭐⭐
- 📖 [GET_COURSE_DATA_NEW.md](GET_COURSE_DATA_NEW.md) - 完整技术文档

## 🚀 快速决策

### 我应该用哪种方式？

#### 使用WebView（方式1）如果：
- ✅ 第一次使用应用
- ✅ 想要快速尝试
- ✅ 没有电脑在身边

#### 使用JSON导入（方式2）如果：
- ✅ WebView提取失败
- ✅ 想要备份数据
- ✅ 需要编辑课程信息
- ✅ 想要分享给同学
- ✅ 跨学期/跨设备使用

## ❓ 问题排查流程

### 问题：WebView无法提取数据

**步骤1：使用调试工具**
```
点击 🐞 按钮 → 查看元素统计
```

**步骤2：判断问题**
- 如果 `.kbcontent: 0` → 不在课表页面或结构变化
- 如果有元素但提取失败 → HTML结构不兼容

**步骤3：切换方案**
```
使用JSON导入（方式2）✅
```

### 问题：JSON导入失败

**步骤1：检查格式**
```
使用 https://jsonlint.com 验证
```

**步骤2：确认完整性**
- 以 `[` 开头
- 以 `]` 结尾
- 包含必需字段

**步骤3：重新提取**
```
重新运行浏览器控制台脚本
```
